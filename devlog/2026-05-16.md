## 备注

### 技术选型调整
- `better-sqlite3` → `sql.js`：前者需要 Visual Studio C++ 编译工具，本机未安装。sql.js 是 SQLite 的 WASM 编译版，纯 JS 无需编译，功能相同。

### Electron 版本
- 最终使用 **Electron 28.3.3**（下载自 npmmirror 镜像）
- **关键发现**：`ELECTRON_RUN_AS_NODE=1` 环境变量导致 Electron 以 Node.js 模式运行，`require('electron')` 返回 exe 路径字符串而非 API 对象。这是本机环境特有的问题，需在每次启动前清除此变量。

### 启动方式
```powershell
Remove-Item Env:ELECTRON_RUN_AS_NODE -ErrorAction SilentlyContinue; npm start
```

### 用户数据路径
- 数据库和设置存储在 `%APPDATA%/clipboard-history/data.db`
- 图片文件将存储在 `%APPDATA%/clipboard-history/images/`
