# 📋 Clipboard History Recorder
一个运行在 Windows 上的轻量级剪贴板历史记录工具，帮你永久保存复制过的所有文字和图片，告别“复制一次就丢”的烦恼。
## ✨ 功能特点
- ✅ **自动记录**：后台静默运行，自动捕获所有复制内容（文字/图片）
- ✅ **历史查看**：按时间倒序展示所有记录，一键快速粘贴
- ✅ **数据持久化**：本地安全存储，重启软件后历史记录不会丢失
- ✅ **轻量无打扰**：界面简洁，不占后台资源，开机可自启
- ✅ **一键复制**：点击任意历史条目，直接复制到剪贴板

---

## 🚀 快速开始

### 方式一：直接使用（推荐）
1.  前往 [Releases 页面](https://github.com/fuzzzzzzzzz/fuz/tree/main) 下载最新版安装包
2.  双击安装并启动软件
3.  软件将自动运行在后台，开始记录你的剪贴板内容

### 方式二：从源码运行（开发/修改用）
需要提前安装 [Node.js](https://nodejs.org/)
