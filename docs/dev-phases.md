# 分阶段执行步骤

## 阶段 1：项目骨架 + 文档体系

**目标**：搭好项目框架，写清所有规范文档

- 初始化 npm 项目，安装 Electron + sql.js
- 创建标准文档（requirements / tech-spec / design-spec / dev-phases / checklist）
- 创建 CLAUDE.md 指引文件
- 创建 .gitignore
- 创建 main.js + preload.js + 空白 index.html
- 验证：`npm start` 打开空白 Electron 窗口

## 阶段 2：数据层

**目标**：数据库能建表、存、查、删、清理

- 实现 database.js（sql.js 封装）
- 数据库初始化（建表）
- CRUD 操作（增删查改）
- 清理逻辑（按天数 + 按数量上限）
- 设置存储与读取
- 验证：插入假数据 → 查询 → 删除 → 清理，每步正确

## 阶段 3：剪贴板监听

**目标**：复制任何文字或图片，自动存入数据库

- 实现 clip​board-monitor.js
- 轮询检测 + hash 去重
- 文字剪贴板保存
- 图片剪贴板保存（转 PNG）
- 集成到 main.js
- 验证：复制文字/图片，检查数据库已写入

## 阶段 4：前端 UI

**目标**：主界面完整可交互

- index.html 页面结构
- style.css 淡红色主题
- app.js 卡片渲染、搜索、置顶、删除
- 设置面板
- IPC 数据联通
- 验证：UI 全部交互正常

## 阶段 5：粘贴 + 托盘 + 开机自启

**目标**：完整闭环

- 点击卡片 → 写剪贴板 → 模拟 Ctrl+V
- 粘贴后窗口隐藏
- 系统托盘 + 右键菜单
- 开机自启注册表
- 验证：点击卡片 → 内容粘贴到记事本 → 窗口隐藏

## 阶段 6：打包 + 验收

**目标**：生成安装程序

- electron-builder 配置
- 应用图标
- 打包 .exe
- 完整验收
- 验证：安装运行，全部功能通过
