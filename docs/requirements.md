# 需求规格说明

## 项目概述

历史粘贴板（Clipboard History）是一款 Windows 桌面应用，自动记录用户复制的文字和图片，方便随时查找和重复使用。

## 功能需求

### FR-1 剪贴板监听
- 系统启动后自动在后台监听剪贴板变化
- 支持文字内容和图片内容
- 去重：连续复制同一内容不重复保存

### FR-2 历史记录存储
- 保留天数可配置：1天 / 3天 / 5天
- 数量上限：默认 200 条
- 超过保留天数或数量上限的旧记录自动清理
- 置顶记录不被自动清理
- 存储持久化，重启软件数据不丢失

### FR-3 历史展示
- 卡片形式排列，最新记录在最上方
- 文字卡片显示内容预览（截断长文本）
- 图片卡片显示缩略图
- 置顶卡片始终在列表顶部，带标记

### FR-4 记录操作
- 置顶：将某条记录固定在列表顶部
- 取消置顶：恢复普通排序
- 删除：移除单条记录（同时删除关联的图片文件）

### FR-5 搜索
- 对文字记录进行关键词搜索
- 实时过滤，输入即搜
- 不区分大小写

### FR-6 粘贴
- 点击卡片 → 内容写回剪贴板 → 自动粘贴到当前活动窗口
- 粘贴后主窗口自动隐藏

### FR-7 系统托盘
- 常驻系统托盘，显示图标
- 左键单击托盘图标：显示 / 隐藏主窗口
- 右键菜单：显示主窗口 / 退出

### FR-8 开机自启
- 可选设置，开启后随 Windows 启动自动运行

### FR-9 设置
- 保留天数选择
- 数量上限调整
- 开机自启开关

## 非功能需求

- 运行时内存占用 < 150MB
- 磁盘占用（不含图片）< 50MB
- UI 响应流畅，卡片列表滚动不卡顿
- 崩溃不影响已存储的数据
